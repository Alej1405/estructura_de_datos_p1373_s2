# Descripcion del Proyecto
Una comparacion sobre dos algoritmos de busqueda.

# Instrucciones para ejecucion
El programa cpompila en python, se requiere la libreria time para visualizar el tiempo de ejecucion y poder realizar la comparacion.

# Breve explicacion.
La busqueda en binario es mucho mas rapida, cuando buscamos numeros pequenos pero se ve una diferencia muy grande cuando los numeros son grandes.
